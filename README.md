## 💻 Tech Stack

*   **Core Library:** React.js (Functional Components, Modular Custom Hooks, and Context State)
*   **Build Environment:** Vite (Providing lightning-fast Hot Module Replacement and production asset bundling)
*   **Styling Architecture:** Tailwind CSS (Utility-first framework utilizing custom fluid spacing, strict responsive breakpoints, and custom typography)
*   **Hosting:** Netlify (Automated builds linked securely with the GitHub main branch)

---

## ✨ Key Features

*   **Interactive Project Showcase:** Dynamic filtering grid showcasing real-world projects with deep-dive technical links.
*   **Modern Interactive UI/UX:** Features fluid component animations, smooth scrolling mechanics, and unified theme parameters.
*   **Fully Responsive Architecture:** Pixel-perfect optimization across smartphones, tablets, laptops, and ultra-wide monitor screens.
*   **Direct Contact Integration:** Clean communication nodes that link directly to professional networks and instant messaging APIs.

## 🛠️ Concepts & Ongoing Full-Stack Journey

As a dedicated developer currently expanding boundaries into backend infrastructure, building this portfolio allowed me to master key client-side engineering practices:
*   **Component Modularity:** Structuring reusable structural UI wrappers (e.g., Project Cards, Form Nodes, Layout Navbars) keeping code thoroughly clean and DRY.
*   **State-Driven Interactivity:** Managing responsive modal behavior and dynamic form tracking using pure React ecosystem values.
*   **Performance Optimization:** Compiling lightweight production code through Vite, leading to near-perfect Core Web Vitals score assessments.
